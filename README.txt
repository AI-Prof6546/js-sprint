JS Sprint — готовый адаптивный сайт-курс по JavaScript.

Что внутри:
- index.html — основная страница сайта
- styles.css — стили
- script.js — логика, темы, практика и прогресс

Как запустить:
1. Распакуй архив.
2. Открой файл index.html в браузере.

Особенности:
- современный адаптивный интерфейс
- программа до блока «Браузер: документ, события, интерфейсы»
- практика, roadmap, FAQ
- сохранение прогресса в localStorage

Важно:
Сайт сделан как оригинальный учебный проект, вдохновлённый идеей структурированного обучения JavaScript,
но без копирования материалов с внешних сайтов.
